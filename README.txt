Java files:
    - Project3.java     - Entry point
    - Cpu.java          - Executes job
    - Job.java          - Data class for job info
    - RRScheduler.java  - Round Robin scheduler
    - SRTScheduler.java - Shortest Remaining time scheduler
    - FBScheduler.java  - Feedback Scheduler
Other files:
    - jobs.txt           - text file with jobs
    - summary.docx       - project summary

Running the program:
    1. Copy all .java and sample (.txt) files to a directory
    2. CD to the directory
    3. Compile with command: javac *.java
    4. Run with commands: java Project3 <RR|SRT|FB|ALL>
